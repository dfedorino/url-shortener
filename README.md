# Основные функции приложения:

1. Реализация алгоритма сокращения и механизма уникальности ссылок для каждого пользователя
   Можно использовать идентификатор длины 6 со следующими кодировкой base64URL (0-9A-Za-z_-) - тогда
   будет 64^6 = 68.7 млрд идентификаторов
2. Реализован переход/редирект по ссылке из консоли
    1. Пользователь вводит длинную ссылку
    2. Сервис возвращает короткую ссылку (по сути идентификатор длинной ссылки)
    3. Пользователь вводит короткую ссылку
    4. Сервис открывает url, переданный пользователем
3. Реализован лимит переходов (подсчёт и блокировка)
   Как только лимит переходов будет исчерпан, пользователь должен быть уведомлен о недоступности
   ссылки, и её использование должно блокироваться.
4. Реализовано время жизни и авто-удаление по сроку
   Время жизни ссылки должно задаваться программно, а не пользователем.
5. Реализованы UUID-идентификация и права на правку/удаление
   UUID пользователя генерируется при первом запросе на создание короткой ссылки и сохраняется для
   последующей работы с его ссылками.
   Все действия по редактированию или удалению ссылок доступны только создателю этой ссылки.
6. Реализация конфигурации приложения из отдельного конфигурационного файла
7. Реализован многопользовательский режим и несколько ссылок у одного пользователя
